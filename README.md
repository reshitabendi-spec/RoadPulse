# 🚦 RoadPulse

**Report. Alert. Prevent.**

RoadPulse is a road safety reporting platform that enables citizens to report unsafe road conditions and helps improve road safety through community participation.

## Features

- User Login & Signup (Firebase Authentication)
- Report road issues with location and description
- Upload road hazard information
- View all submitted reports
- Road safety awareness section
- Simple and responsive interface

## Types of Issues Supported

- Unsafe Speed Breakers
- Potholes
- Missing Zebra Crossings
- Uneven Roads
- Missing Sign Boards
- Improper Truck Parking

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Firebase Firestore

##  How to Run

1. Download or clone the repository.
2. Open the project in Visual Studio Code.
3. Configure Firebase credentials.
4. Open `index.html` using Live Server.
5. Login and start reporting road hazards.

##  Objective

To encourage citizens to actively participate in reporting road hazards and improve road safety through a simple web-based platform.
